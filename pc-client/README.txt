How to use the source code?
-----------------------------------
The 3d-scanner-win32 only contains partial RoboPeak 3D-scanner PC-client sourcecode.
Since the full source contains RoboPeak property code, such code has been removed before releasing.

This means you need to provide similar implementation to the missing parts in order to compile the source code. 

But don't worry, the missing code is just related to some utility functions. They should be implemented easily.

Meanwhile, you may try the precompiled demo in the precompiled-demo folder.

========================================

如何使用源代码?
-----------------------------------
在3d-scanner-win32中仅包含了RoboPeak 3D-scanner PC客户端的部分源代码。
由于完整的代码包含了RoboPeak尚未公开的代码，这些代码在发布时已经移除了。

这就意味着您需要提供这些缺少部分的类似实现来保证代码的编译。

但不用过于担心，有这些遗漏的代码仅仅是一些辅助函数。他们很容易实现。

同时，您也可以尝试precompiled-demo目录下的预编译demo。
